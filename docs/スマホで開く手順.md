# スマホで畑アプリを開く手順

## まずPC側で起動

1. PCでこのフォルダを開く。
2. `tools/start-mobile-server.cmd` をダブルクリックする。
3. 黒い画面が開きます。
4. Windowsの許可画面が出たら、`プライベートネットワーク` を許可する。
5. 黒い画面に表示された `スマホで開くURL候補` を確認する。
6. 候補が複数ある場合は、`Wi-Fi` と表示されているURLを使う。

例:

```text
http://192.168.10.104:8080/  (Wi-Fi)
```

## スマホ側で開く

1. スマホをPCと同じWi-Fiにつなぐ。
2. スマホのブラウザで、PC側に表示されたURLを開く。
3. 表示できたら、ブラウザのメニューからホーム画面に追加する。

## iPhoneの場合

1. SafariでURLを開く。
2. 共有ボタンを押す。
3. `ホーム画面に追加` を押す。

## Androidの場合

1. ChromeでURLを開く。
2. 右上メニューを押す。
3. `ホーム画面に追加` または `アプリをインストール` を押す。

## データ同期について

PCとスマホで同じデータを使うには、Supabase同期設定が必要です。

PC側で:

1. アプリ右上の `☁` を開く。
2. Supabase URL、anon key、共有データIDを設定する。
3. `クラウドへ保存` を押す。
4. `設定リンクをコピー` を押す。
5. そのリンクをスマホへ送る。

スマホ側で:

1. 送られてきたリンクを開く。
2. 同期設定が取り込まれたら `クラウドから取得` を押す。

## うまく開けないとき

- PCとスマホが同じWi-Fiか確認する。
- Windowsのファイアウォールで許可したか確認する。
- 黒い画面を閉じていないか確認する。
- URLの数字がPC側に表示されているものと同じか確認する。
- `vEthernet` や `Default Switch` と表示されたURLではなく、`Wi-Fi` のURLを使う。
- スマホがモバイル回線ではなくWi-Fiになっているか確認する。
